v0.2.3
Released 2017-06-29

0.2.3 is a version of Super Mario Scralaxy. it was released on 29th June 2017. Since 0.2.2 was released around the same time, and this project was supposed to be 0.2.2, the later project became version 0.2.3. However, since the project was not made with the 0.2.2 engine, the new elements of version 0.2.2 did not make the cut.

Features

  • New swimming script
  • Ice function
  • Flashing glitch fixed
  • Cursor layering fixed
  • Pipes
  • Mystery box
  • Menu background
  • New text
  • Lava
  • New levels
  • New menu
  • Title screen animation
  • Underground levels
  • More dialogue
  • Block breaking reworked
  • Smoother sprites and scripts
  • Fire physics
  • Removed old levels
